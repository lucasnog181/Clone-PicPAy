### Como executar:
Execute os seguintes comandos em um terminal:
<br>
<br>
É necessário ter o expo instalado:
```
npm install --global expo-cli

```
<br>

## Após instalar o Expo:

Execute os seguintes comandos em um terminal:

```
yarn start
```

```
npm start
```
